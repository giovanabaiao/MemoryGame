# Fonts

The game tries these font paths in order:

1. `assets/fonts/PressStart2P-Regular.ttf`
2. `assets/fonts/VT323-Regular.ttf`
3. `assets/fonts/font.ttf`
4. Common system fonts on macOS/Windows

For consistent pixel-art style, add your chosen pixel font file here.  
Recommended: `PressStart2P-Regular.ttf` (or any OFL-compatible bitmap/pixel font).
